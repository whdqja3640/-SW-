먼저 실행 시, LoginWindow.java 파일을 실행하시면 됩니다.


테스트 계정을 각 역할군마다 생성해놓았습니다. (MemeberInfo/member.txt 내에 계정 정보 존재)
- ID, PW, 역할군 순서로 적어두겠습니다.
  customer, customer, Customer
  admin, admin, Administrator
  farmer,farmer, Famer (농장 등록까지 마친 계정입니다)

회원가입
 - 등록된 계정이 아닌 자신만의 계정을 이용하고자 한다면 수행하면 됩니다.
    - 이때, ID는 중복된 값이 불가능합니다. 또한, PW 확인 과정을 제대로 수행해야 회원가입에 성공할 수 있습니다.

Customer 역할군
 - 농장 검색 클릭 후, 원하는 기준 선택하여 검색을 할 수 있습니다.
    - 입력 창에 입력 후 검색 버튼 클릭하면 됩니다.
      - 원하는 농장을 선택 후, 농장 보기 클릭을 하면 해당 농장에서 예약 및 판매가 가능한 농작물 목록을 열람할 수 있습니다.

 - 예약 정보 버튼을 클릭하여 자신의 예약 정보를 볼 수 있습니다.

Farmer 역할군
 - 가입 후 농부 인증을 거치지 않을 시 정상적으로 기능을 이용할 수 없습니다.
   - 농부 인증을 해둔 후, 관리자가 승인할 시, 정상적으로 이용이 가능합니다.

 - 예약 정보는 자신의 농장에 예약된 농작물 내역을 출력합니다.

 - 농장 정보 보기를 통해 재배 중인 농작물, 판매 중인 농작물 그리고 저장고 내역을 확인할 수 있습니다.
   - 재배 중인 농작물의 경우 추가, 삭제, 수정, 수확 시 올바른 값을 입력하면 서버에 해당 내역이 반영됩니다.
      - 수확은 해당 내역을 수정한 채로 서버에 저장이 되므로 신중히 사용하시면 됩니다.

   - 판매 중인 농작물의 경우 예약 가능 리스트와 구매 가능 리스트를 클릭 시 해당하는 내역이 출력됩니다.
      - 구매 / 예약 시 올바른 정보를 입력 후 요청을 하면 해당 내역이 서버에 반영됩니다.

   - 저장고 내역의 경우, 자신의 농장에 있는 내역을 출력하며 추가, 수정 및 삭제를 수행할 수 있습니다.
   
Administrator 역할군
 - 농부 회원 승인 요청이 있을 시, 해당 내역에 출력되며 승인 및 거절을 수행할 수 있습니다.
